# [PRD] 소상공인 수익 분석 서비스 - 간편 모드 (Lite Mode)

## 1. 단계별 사용자 경험 (UX) 전략

### [Step 1] 흥미 유발 (Lite Mode)

사용자가 서비스의 가치를 즉각적으로 느낄 수 있도록 최소한의 입력값으로 시각적 보상을 제공합니다.

- **핵심 입력 항목 (3가지 영역)**:
  1.  **기본 정보**: 업종 선택, 월 영업일수, 목표 순이익 설정.
  2.  **판매 현황**: 대표 메뉴명, 판매 단가, 하루(또는 월) 판매량 입력.
  3.  **대략적 비용**: "한 달에 대충 얼마 쓰시나요?"에 대한 총비용 (변동비+고정비 합계액) 입력.

### [Step 2] 가시적인 결과 제공 (Wow Moment)

입력 즉시 복잡한 계산 과정을 거쳐 다음과 같은 직관적인 리포트를 제공합니다.

- **수익 분석 카드**: "현재 사장님의 월 예상 순이익은 **OOO원**입니다."
- **목표 달성 가이드**: "목표인 OOO원까지 **OOO원**이 부족합니다."
- **실전 액션 아이템**: "메뉴 A를 하루에 **N개**만 더 팔면 목표 수익에 도달할 수 있습니다!"

---

## 2. 전문가 모드 유도 전략 (Natural Hook)

단순 계산 결과에 그치지 않고, 데이터의 정밀도를 높여 실제 경영에 도움이 되는 '전문가 모드'로 자연스럽게 전환하도록 유도합니다.

- **Hook 1: 비용 효율 분석**
  - "단순 통계 비용만으로는 누수되는 고정비를 찾기 어렵습니다. 인건비와 임차료를 상세히 나누어 분석하면 효율적인 비용 절감 포인트를 짚어드립니다."
- **Hook 2: 금융 지표 연동**
  - "대출 금리 변동에 따른 리스크를 알고 싶으신가요? 실제 투자금과 대출 이자를 반영하여 감가상각까지 고려한 '진짜 마진'을 확인하세요."
- **Hook 3: 업종별 특화 분석**
  - "제조업인가요? 공장 경비와 매입 원가를 분리하면 정확한 제조원가 명세서와 손익분기점을 도출할 수 있습니다."

---

## 3. 간편 모드 데이터 구조 (JSON Schema)

간편 모드에서는 다음과 같이 정규화된 최소 데이터셋을 생성하여 서버에 저장합니다.

```json
{
  "basic": {
    "industryCd": "3",
    "workDay": 25,
    "monTargetPrice": 5000000,
    "saleMode": "1"
  },
  "sale": {
    "menu": [{ "menuTitle": "아메리카노", "menuPrice": 4500, "menuQty": 50 }]
  },
  "liteCost": {
    "totalMonthlyCost": 3500000
  }
}
```

---

## 4. 전문가 모드 전환 시 데이터 확장 (Expansion)

사용자가 '전문가 모드'로 전환하면 기존의 간편 데이터를 바탕으로 상세 분석 필드들을 확장합니다.

| 구분               | 확장되는 가치 및 보상                                                                    |
| :----------------- | :--------------------------------------------------------------------------------------- |
| **인건비 상세화**  | 제조/판매 부문별 노무비 비율과 4대 보험 부담금 분석 가시화                               |
| **투자/자금 정보** | 자본금 및 대출금 입력 시 이자 비용과 감가상각비 자동 반영                                |
| **비용 상세화**    | 임차료, 수도광열비 등 24종 이상의 항목에 대해 '고정액' 또는 '매출 대비 %' 기준 설정 가능 |
| **다년도 추정**    | 3개년 매출 성장률 및 비용 상승률을 반영한 미래 손익 및 재무상태표 예측                   |

### 전환 로직 (Implementation Note)

1.  **Data Mapping**: `liteCost.totalMonthlyCost` 데이터를 기반으로 전문가 모드의 기본 비용 항목들을 초기화합니다.
2.  **UI Feedback**: 전문가 모드로 전환 시, 확장된 입력 항목(인건비, 투자금 등)에 대한 가이드 툴팁과 안내 메시지를 노출하여 거부감을 최소화합니다.
